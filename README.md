# fast-task-cli

Fast and minimal CLI task manager focused on speed and simplicity.

## ⚡ Features

* Ultra-fast task input (`task+2`)
* Quick delete with `-index`
* Colored deadlines (urgent / soon / normal)
* Minimal interface — no unnecessary steps
* Local file storage (no setup needed)

---

## 🚀 Usage

Run the script:

```bash
python TaskMGR.py
```

---

## ⚡ Fast Input Examples

```text
math+2
biology+1homework
-2
```

* `+N` → adds deadline in N days
* `-index` → deletes task instantly

---

## 🎯 Philosophy

This tool is built for **speed**:

* minimal typing
* no menus
* no distractions

> Think → type → done

---

## 📦 Storage

Tasks are saved in:

```
task.txt
```

Format:

```
name | deadline | description
```
